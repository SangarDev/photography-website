# Photography Website

A simple **HTML + CSS photography website** project with sidebar navigation and social icons.  
Built to practice responsive layouts, sidebars, and icon integration.

## Features

- Sidebar menu with toggle
- Responsive design
- FontAwesome icons
- External fonts (Google Fonts)
- Background image overlay
- Social media links

## Tech Stack

- HTML5
- CSS3
- FontAwesome Icons
- Google Fonts
